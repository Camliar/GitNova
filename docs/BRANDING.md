# GitNova Brand Guideline

## 品牌核心

- **产品名：** GitNova
- **英文 Slogan：** Smarter Git. Deeper Insight.
- **中文表达：** 更聪明的 Git 客户端，洞察每一次提交背后的故事。
- **品牌性格：** 清晰、克制、可信、敏锐。

“Nova”表达从复杂历史中显现的新线索；视觉符号用分叉轨迹与星芒表达 Git 历史和洞察。产品定位见[项目说明](PROJECT.md)，语气应服务于[产品愿景](VISION.md)。

## 品牌色

| Token | Hex | 用途 |
| --- | --- | --- |
| Nova Indigo | `#645CFF` | 主品牌、主要操作、焦点 |
| Insight Cyan | `#34D9E8` | 洞察、高亮、轨迹 |
| Deep Space | `#111426` | 深色背景、主文字 |
| Orbit Slate | `#667085` | 次级文字、边框 |
| Starlight | `#F7F8FC` | 浅色背景、深色界面文字 |

正文与交互色必须满足 [UI Guideline](UI_GUIDELINE.md) 的 WCAG 2.2 AA 对比度要求；品牌色不是状态色的唯一表达。

## Logo 使用规范

- 首选 [`assets/logo/gitnova-logo.svg`](../assets/logo/gitnova-logo.svg)；紧凑空间使用 [`assets/icons/gitnova-mark.svg`](../assets/icons/gitnova-mark.svg)。
- Logo 四周至少保留等于图标内圆直径一半的净空。
- 标准组合最小宽度为 160 px，图标最小尺寸为 24 px。
- 仅使用提供的全彩或单色版本；不可拉伸、旋转、改色、描边、加阴影或重排。
- 复杂背景上应放在 Deep Space 或 Starlight 的纯色承载面。
- 当前所有文件均为占位资产，不应注册商标或用于正式发行。

源文件、PNG 导出和颜色令牌位于 [`assets/`](../assets/README.md)。产品内视觉规则见[界面指南](UI_GUIDELINE.md)。
